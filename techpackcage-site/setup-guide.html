Section 02
Week 1 — Foundation
Domain, hosting, WordPress, WooCommerce, Cloudflare, SSL. Complete these steps in order. Takes 2–4 hours total.
Day 1 — Domain + Hosting
1
Purchase domain at Namecheap
Go to namecheap.com → Search bar → type "TechPackCage.com" → Add to cart → Checkout. Enable "WhoisGuard" (free — hides your personal address from WHOIS lookup). Annual registration.
namecheap.com
If TechPackCage.com is taken, try: TechPackCage.ca / ThePetTechStore.com / PetTechPack.com
2
Sign up Hostinger Business Shared Hosting
Go to hostinger.com → Hosting → Shared Hosting → Business plan ($3.99/mo) → Choose 12-month billing → Add to cart → Complete purchase. Business plan required for WooCommerce (more resources than Starter).
hostinger.com/hosting/wordpress
3
Point Namecheap DNS to Hostinger
In Hostinger hPanel → Hosting → Manage → scroll to "Nameservers". Copy the 2 nameservers shown. Go to Namecheap → Domain List → Manage → Nameservers → Custom DNS → paste Hostinger nameservers. DNS propagation takes 1–24 hours.
Hostinger nameservers format: ns1.dns-parking.com ns2.dns-parking.com (exact values shown in your Hostinger hPanel)
4
Enable SSL (Let's Encrypt)
Hostinger hPanel → Security → SSL → Select your domain → Let's Encrypt → Install. This gives you https://TechPackCage.com for free. Auto-renews every 90 days.
Enable SSL BEFORE installing WordPress to avoid mixed-content issues.
Day 2 — WordPress + WooCommerce
5
Install WordPress via Hostinger hPanel
hPanel → Website → WordPress → Install. Select your domain. Set admin username + strong password. Email: franciscoderek7@gmail.com. IMPORTANT: Save your WP admin credentials somewhere safe.
hpanel.hostinger.com
6
Install WooCommerce plugin
WP Admin (yoursite.com/wp-admin) → Plugins → Add New → search "WooCommerce" → Install Now → Activate. Run the setup wizard: currency (CAD), store location (Canada), product type (Physical), skip payment for now.
wordpress.com/plugins/woocommerce
7
Install Astra theme
WP Admin → Appearance → Themes → Add New → search "Astra" → Install → Activate. Then: Appearance → Customize → paste the TechPackCage WooCommerce CSS (see Section 07) into Additional CSS → Publish.
wpastra.com
8
Set up Cloudflare CDN (strongly recommended)
cloudflare.com → Add a site → enter TechPackCage.com → Free plan → Follow DNS migration steps → Update Namecheap nameservers to Cloudflare's. Enables global CDN, DDoS protection, and performance gains for free.
cloudflare.com
After Cloudflare: enable "Speed → Optimization → Auto Minify" (JS, CSS, HTML) and "Caching → Browser Cache TTL → 1 month"
9
Install LiteSpeed Cache
WP Admin → Plugins → Add New → search "LiteSpeed Cache" → Install → Activate → Settings → enable page caching. This is ESSENTIAL on shared hosting — reduces load time from 4–8 seconds to under 2 seconds.
Section 03
Week 2 — Products
AliDropship installation, product import, pricing rules, categories, and shipping zones. By end of this week you should have 100+ live products.
Day 8-9 — AliDropship Setup
1
Purchase AliDropship plugin ($89 one-time)
alidropship.com → AliDropship WooCommerce Plugin → Purchase. One-time payment. No recurring fees. You get a zip file to upload to WordPress. This covers unlimited products and orders.
alidropship.com
2
Install AliDropship via WordPress
WP Admin → Plugins → Add New → Upload Plugin → Choose File → select the AliDropship zip you downloaded → Install Now → Activate. Then enter your license key from the purchase email.
Also install the "AliDropship Helper" Chrome extension — required to import products directly from AliExpress product pages.
3
Configure global pricing rules (2.5× markup)
AliDropship → Settings → Pricing → Pricing Formula → set to "Multiply by 2.5". Also set: Min price = $9.99, Round prices to $X.99 format (psychological pricing). This sets every product automatically.
Pricing Formula: [COST] * 2.5 Min Markup: $5.00 Price Rounding: Round to X.99 Example: AliExpress cost $14.00 → Sell price: $34.99
Day 9-12 — Import Products
4
Import first 25 products: GPS Trackers
Go to AliExpress → search "GPS dog tracker 4G" → filter: 4★+ rating, 500+ orders → click product → AliDropship Chrome extension → Import → edit title, set category. Repeat for 25 trackers.
aliexpress.com → search: "GPS dog tracker waterproof 4G"
5
Import 25 Auto Feeders & 25 Pet Cameras
Repeat the import process. Search terms: "automatic cat feeder camera wifi" and "pet camera treat dispenser". Filter: 1000+ orders, 4.5★+. These are your highest-margin, best-selling categories.
aliexpress.com → "automatic pet feeder camera" / "360 pet camera treat"
6
Import 25 more across remaining categories
5 products each from: AI Toys, Self-Cleaning Litter, Smart Beds, Health Monitors, Training Tech. Focus on items with 500+ orders and clear product photos — these convert better.
7
Set up 15 WooCommerce product categories
WP Admin → Products → Categories → Create all 15 categories from the site plan. Add category descriptions (good for SEO). Upload a 600×600 category image for each.
GPS & Trackers | Auto Feeders & Waterers | Invisible Fences Health Monitors | AI & Interactive Toys | Smart Beds Auto Litter & Waste | Training Tech | Safety Gear Pet Cameras | Wearable Fitness | Grooming Tech Travel & Mobility | Aquatic Tech | Small Animal & Exotic
8
Configure WooCommerce Shipping Zones
WooCommerce → Settings → Shipping → Shipping Zones → Add Zone. Create 3 zones:
Zone 1: Canada - Free shipping: orders ≥ $50.00 - Flat rate: $5.99 (orders under $50) - Express Job Shipping: $12.99 (custom method - see Section 09) Zone 2: United States - Flat rate: $7.99 - Express: $15.99 Zone 3: International (Rest of World) - Calculated: based on weight / supplier
Section 04
Week 3 — Payments & Marketing
Connect Stripe, configure PayPal, install SEO, set up email marketing, Facebook Pixel, and social channels.
Day 15-16 — Payments
1
Connect Stripe to WooCommerce
WooCommerce → Settings → Payments → Stripe → Enable → Manage. Enter your Stripe API keys from dashboard.stripe.com → Developers → API Keys. Enable Apple Pay and Google Pay (free — toggle in Stripe settings).
dashboard.stripe.com → Developers → API Keys
Use TEST mode first. After placing a test order successfully, switch to LIVE keys.
2
Enable PayPal
WooCommerce → Settings → Payments → PayPal Standard → Enable. Enter your PayPal email: franciscoderek7@gmail.com. Customers who prefer PayPal will see it at checkout automatically.
Day 16-19 — SEO + Marketing
3
Install RankMath SEO (free)
Plugins → Add New → "RankMath SEO" → Install → Activate → Run Setup Wizard. Connect to Google Search Console (free) for indexing. Set: Homepage title "TechPackCage — Best Pet Tech Prices | GPS Collars, Smart Feeders & More".
4
Set up Mailchimp (free to 500 subscribers)
mailchimp.com → Create free account → Audience → Settings → Signup forms. Install WooCommerce Mailchimp plugin: Plugins → Add New → "Mailchimp for WooCommerce" → connect your account. This auto-tags buyers for abandoned cart emails.
mailchimp.com
5
Install Abandoned Cart Recovery
Plugins → Add New → "WooCommerce Abandoned Cart Lite" → Install → Activate. Configure: send reminder 1 hour after abandonment, 24 hours, 72 hours. Subject line: "Still thinking about it? Your cart is waiting 🐾"
6
Set up Facebook Pixel
business.facebook.com → Events Manager → Connect Data Sources → Web → Facebook Pixel → follow setup. Install "Pixel Your Site" WordPress plugin → paste Pixel ID. Enables Facebook ad retargeting from day one.
business.facebook.com/events_manager
7
Create social media accounts
Create: Facebook Page (TechPackCage), Instagram (@techpackcage), TikTok (@techpackcage). Use the same green/gold branding. Profile photo: TechPackCage logo (CSS-generated or simple text logo). Post product videos immediately.
TikTok is your highest ROI channel for pet products. 15-second product demos with pets using the tech go viral constantly. Start immediately.

Section 09
Job Shipping Integration
Add "Express Job Shipping" as a custom WooCommerce shipping method. PHP code to paste into your child theme's functions.php or a custom plugin file.
/** * TechPackCage — Job Shipping Custom Shipping Method * Add to: wp-content/themes/astra-child/functions.php * OR create: wp-content/plugins/tpc-job-shipping/tpc-job-shipping.php */ add_filter( 'woocommerce_shipping_methods', function( $methods ) { $methods['tpc_job_shipping'] = 'TPC_Job_Shipping_Method'; return $methods; }); add_action( 'woocommerce_shipping_init', function() { class TPC_Job_Shipping_Method extends WC_Shipping_Method { public function __construct( $instance_id = 0 ) { $this->id = 'tpc_job_shipping'; $this->instance_id = absint( $instance_id ); $this->method_title = 'Express Job Shipping'; $this->method_description = '2-3 day delivery via local independent drivers'; $this->supports = [ 'shipping-zones', 'instance-settings' ]; $this->title = '⚡ Express Job Shipping (2-3 days)'; parent::__construct( $instance_id ); } public function calculate_shipping( $package = [] ) { $this->add_rate([ 'id' => $this->get_rate_id(), 'label' => '⚡ Express Job Shipping — 2-3 Business Days', 'cost' => 12.99, 'meta_data' => [ 'delivery_type' => 'local_driver', 'estimated_days' => '2-3', ], ]); } } }); // end woocommerce_shipping_init /** * Send driver assignment notification on Job Shipping order * Fires when order status changes to "processing" */ add_action( 'woocommerce_order_status_processing', function( $order_id ) { $order = wc_get_order( $order_id ); if ( ! $order ) return; foreach ( $order->get_items( 'shipping' ) as $item ) { if ( $item->get_method_id() === 'tpc_job_shipping' ) { // Notify dispatch email — replace with your driver dispatch system wp_mail( 'dispatch@techpackcage.com', 'New Job Shipping Order #' . $order_id, 'Order #' . $order_id . ' — Ship to: ' . $order->get_formatted_shipping_address() . '\n\nItems: ' . implode( ', ', array_map( fn($i) => $i->get_name(), $order->get_items() ) ) ); } } });
Phase 2 Upgrade: Replace the wp_mail() with a Google Maps Route Optimization API call to auto-assign the nearest available driver. Use Stripe Connect to split payment: 80% to driver, 20% to TechPackCage. Build a simple driver app with Google Maps SDK.
