## TRADEMARK WATCH SERVICES

Once marks are filed:
1. **CIPO watch:** Set up Google Alerts for each mark name + competitors
2. **Formal watch service:** TrademarkNow, CompuMark, or IP Toolworks (~$50-150/mark/year)
3. **Domain watch:** Porkbun/Namecheap domain alert services for similar TLDs

## BRAND CONFLICT PRE-SEARCH CHECKLIST

Before filing, run these searches:
- [ ] CIPO Trademarks Database: trademarks.ic.gc.ca
- [ ] USPTO TESS: tmsearch.uspto.gov
- [ ] WIPO Global Brand Database: branddb.wipo.int
- [ ] Google (check existing businesses with same name)
- [ ] Domain availability for primary TLDs

*Run these searches for each Priority 1 mark before investing in CIPO filing fees.*
