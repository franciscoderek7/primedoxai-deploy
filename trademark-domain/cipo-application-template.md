## FILING PROCESS (CIPO Online)

1. Go to: **www.ic.gc.ca** → Trademarks → Apply
2. Create a GC Key account (or sign in if you have one)
3. Select "Online application — Trademark"
4. For each mark: complete applicant info, mark description, goods/services, and basis
5. Pay by credit card ($458 CAD per mark, first class; $161 per additional class)
6. Keep your application number — CIPO will send it by email
7. Examination takes 6-24 months; respond to any objections (examiner's reports) within the deadline

**Recommended:** File all Priority 1 marks in one session to minimize repeat administration overhead.

## USPTO TEAS PLUS — KEY DIFFERENCES FROM CIPO

1. Must identify goods/services using the ID Manual (not free-form text like CIPO)
2. Must provide a "specimen" showing the mark in use (screenshot of website, product image, etc.)
3. Basis: either use in commerce (Section 1(a)) or intent to use (Section 1(b))
4. File at: teas.uspto.gov
5. Recommended basis for most marks: 44(e) — Based on foreign (Canadian) registration — use this AFTER CIPO grants the Canadian registration

**For cannabis-related marks (CCLDR, DOC WEEDLAW):** USPTO may refuse registration because cannabis-related goods/services are not lawful under federal US law. Consider filing only the non-cannabis-specific services (e.g., "general constitutional law education") under a separate application, or skip USPTO for the cannabis-specific marks until US federal law changes.
