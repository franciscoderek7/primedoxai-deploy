BENO-X: Bad Exemption = No Offense

The core principle: if the exemption framework is constitutionally invalid — if the regulatory structure that defines the offense is itself in breach of the Charter — then the underlying conduct cannot constitute a criminal offense. This principle does not eliminate law. It requires that law be constitutionally coherent. A prohibition built on a broken exemption system is a prohibition without a valid foundation.
