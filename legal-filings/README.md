## Key rules (Ontario)
- Rule 19.02: Defendant may be noted in default after 20 days from service of Statement of Claim
- Rule 19.04: Default judgment by registrar (liquidated amounts only)
- Rule 19.05: Default judgment by court (unliquidated / assessed damages — requires motion)
- Section 103 CJA + Rule 42: CPL — requires a reasonable claim to an interest in land
