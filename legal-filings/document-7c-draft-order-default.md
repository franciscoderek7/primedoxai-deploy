## FILING INSTRUCTIONS FOR DRAFT ORDER

1. Attach to your motion record as the last document (after supporting affidavit)
2. Bring **two clean copies** to the motion: one for the judge/registrar to sign,
   one for the court file
3. After the order is granted, obtain a certified copy from the court registrar
4. File the certified copy with the Sheriff's office (Writ of Seizure and Sale)
5. If property is involved, register the order (or a notice of it) at the
   Land Registry Office

---

## IF THE REGISTRAR (NOT A JUDGE) SIGNS:
- Rule 19.04 allows the registrar to sign default judgment only for **liquidated claims**
  (a definite sum of money already established, not requiring assessment)
- If your $1M claim is from a definite agreement/promissory note/debt, it is liquidated → registrar can sign
- If damages require assessment (tort, breach of fiduciary duty, etc.) → you need a judge under Rule 19.05
- Clarify with the court registrar which path applies to your specific claim
