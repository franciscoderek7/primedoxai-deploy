Chrome Web Store — OmniaGuard Shield Submission

Complete step-by-step guide. Total time: ~20 minutes. Cost: $5 USD one-time.
1 Pay the $5 Developer Registration Fee

You only pay this ONCE. It covers all future extensions.

1. Go to: chrome.google.com/webstore/devconsole

2. Sign in with your Google account

3. Click "Pay $5.00 registration fee"

4. Complete payment (credit card)

5. You now have developer access — keep this tab open

2 Create the Extension ZIP File

The Chrome Web Store requires a .zip file containing your extension.

1. Download your repo from GitHub: github.com/franciscoderek7/primedoxai-deploy

2. Navigate to the omniaguard-extension/ folder

3. Create icons first — open create-icons.html in Chrome, click each icon to download icon16.png, icon48.png, icon128.png → place in icons/ folder

4. Select ALL files inside omniaguard-extension/ (not the folder itself)

5. Right-click → Compress / Create ZIP → name it omniaguard-shield-v1.0.0.zip

⚠️ Do NOT zip the folder itself — zip the CONTENTS. The manifest.json must be at the root of the zip, not inside a subfolder.
3 Create New Item in Developer Console

1. In the Developer Console, click "+ New Item"

2. Upload your omniaguard-shield-v1.0.0.zip

3. Chrome will parse the manifest and show you the listing editor

4. Fill in all fields using the copy below

6 Submit for Review

1. Fill in all required fields (marked with *)

2. Set visibility to Public

3. Set distribution to All regions

4. Click "Submit for Review"

Review timeline: Usually 1-3 business days. First submission sometimes takes up to 7 days.

Once approved: Extension goes live at chrome.google.com/webstore/detail/omniaguard-shield/[ID]

Your extension ID will be assigned automatically — use it on the OmniaGuard website to link to the extension.
