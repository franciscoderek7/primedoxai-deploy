★
How to add DNS records for one domain (repeat for each)
DNS
1
At Porkbun, click DNS next to your domain name
2
Delete any existing A records for @ (the root) — they'll conflict
3
Add these 4 A records — one at a time. Type: A · Host: @ · Answer: (IP) · TTL: 600
GitHub Pages IP addresses (add all 4):
A Host: @ → 185.199.108.153
A Host: @ → 185.199.109.153
A Host: @ → 185.199.110.153
A Host: @ → 185.199.111.153
4
Add ONE CNAME record: Type: CNAME · Host: www · Answer: franciscoderek7.github.io · TTL: 600
CNAME Host: www → franciscoderek7.github.io
5
Back in GitHub → repo Settings → Pages → scroll to Custom domain → type your domain (e.g. ccldr.net) → click Save. GitHub handles the HTTPS cert automatically.
6
DNS propagates in 5–30 minutes. You're done with that domain.
