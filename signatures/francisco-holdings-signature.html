📋 How to install in Gmail:
1. Click inside the signature box below and press Ctrl+A (Mac: Cmd+A) to select all
2. Press Ctrl+C (Mac: Cmd+C) to copy
3. Open Gmail → Settings (⚙) → See all settings → Signature → New
4. Click in the signature editor and press Ctrl+V (Mac: Cmd+V) to paste
5. Save changes
