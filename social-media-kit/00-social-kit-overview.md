## HOW TO USE THIS KIT

Each brand's profile spec (in `loop-a-profiles.md` and `loop-b-profiles.md`) contains:
- Username/handle suggestions
- Bio copy (under character limits for each platform)
- Profile photo description (what to put there — you supply the actual image)
- Cover photo/banner description
- Brand color hex codes
- Link-in-bio URL
- Pinned post copy

Load each spec into your Buffer, Hootsuite, or Later account for scheduling.
The 30-day content calendars are in separate files.
