📧 How to Add These to Gmail
Step 1: Open Gmail → Click ⚙️ Settings (top right) → See All Settings
Step 2: Under "General" tab → scroll to "Signature" section → click "Create new"
Step 3: Name it (e.g. "OmniaGuard") → Click "More options" (⋮) → Click "Insert signature"
Step 4: In the editor → click the source/HTML icon (<>) if available, or use the "Insert signature" toolbar
Step 5: BEST METHOD: Copy the HTML → Open Gmail Compose → Paste directly (Gmail accepts HTML from clipboard)
Step 6: Set "Default signature for new emails" and "Default signature for replies/forwards" per email address

For multiple Gmail accounts: Each signature must be configured under the account it belongs to.
Use Gmail's "Send mail as" feature (Settings → Accounts → Send mail as) to add business emails to your main Gmail account.
