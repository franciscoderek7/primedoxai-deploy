How This Detects VPN
🔬
WebRTC IP Scan
Reads ALL local IP addresses via WebRTC, including VPN tunnel IPs (e.g. 10.8.x.x from OpenVPN, WireGuard addresses). Normal WiFi only shows your router IP (192.168.x.x).
🌐
ISP Fingerprinting
Checks if your public IP's ISP name contains known VPN provider keywords (NordVPN, Mullvad, ProtonVPN, Cloudflare WARP, etc.).
⚠️
WebRTC Leak Detection
If a VPN is active but WebRTC still exposes a public IP, your real IP is leaking. This is a critical security failure for privacy-focused VPN users.
🔄
Baseline Comparison
Stores your first-check IP as baseline. If IP changes on subsequent checks, alerts you — this catches VPN connect/disconnect events.
Browser Limitation: This tool uses available browser APIs. For full network traffic inspection (packet analysis, DNS monitoring, firewall rules), a native Android VPN service is required. See the Android APK build guide for native capabilities.
DNS Consistency Check
Queries 3 independent IP-detection services. If all agree → no DNS split. If any differ → possible DNS leak or split horizon.
Click "Run DNS Check" to test DNS consistency.
